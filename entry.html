Network & script inspection tool

  1. Navigate to the site you want to capture (type the URL in the address bar or use a bookmark).
  2. Use the site as usual (log in, click around, load the pages you care about). All requests and script responses are recorded.
  3. When finished, switch to the terminal and press Enter to save and close.

Output is written under output/<hostname>/:

  • networkcalls.json – all requests and responses
  • scripts.json – JavaScript responses only

Session is saved to session.json so the next run can reuse cookies/storage. Use --no-session to start fresh.

To force output to a specific hostname, run with --url https://example.com. To limit which hosts are captured, use --domains example.com,api.example.com.
